Full-Stack Web Development

My latest adventure into learning a new skill has brought me to Web Development, understanding HTML, CSS, JavaScript, React and more. In the future I also plan to learn PHP and MySQL.

What is Full-Stack to me?

Computers have always been a peek interest of mine, from building them, to using them. I have always dreamed of a career in the tech industry. Full Stack is my chance to flex my computer skills to a new level and open doors to a new career path where I take my dreams and make them reality. As a Full-Stak Web Developer I will be able to accomplish these goals by having a deeper understanding of how to program web pages through the use of HTML, CSS, JavasScript and other programming languages.

A Full-Stack Web Developer works on both the front-end and back-end portions of the web, creating not just the code but styling and designing the web page and applications used within. Being a "Universal Soldier" has always been my ambitions, I do specialize in a few areas, however I enjoy bing useful in many areas. Full-Stack is an opportunity to broaden my abilities and create something I can be proud of
